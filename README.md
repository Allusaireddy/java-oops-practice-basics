Bank Account Management System (Java)
A core Java application demonstrating fundamental Object-Oriented Programming (OOP) principles, including classes, objects, and state management.

🚀 Features
Account Initialization: Create new accounts with a holder name, account number, and initial balance.

Transaction Management: Securely deposit and withdraw funds with real-time balance updates.

Validation Logic: Built-in checks to prevent withdrawals that exceed the current balance.

Data Display: Quick console output for account status and holder details.

🛠️ Technical Stack
Language: Java

Environment: Eclipse IDE

Version Control: Git & GitHub

📂 Project Structure
BankAccount.java: Contains the logic for account operations (the "Blueprint").

BankApp.java: The entry point (Main class) that simulates real-world usage.
